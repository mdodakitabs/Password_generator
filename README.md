A simple web application that generates strong, random passwords.

## Features
* Generates passwords with a mix of uppercase, lowercase letters, and numbers
* Password length is fixed at 8 characters
* Password is displayed in a text field with blue text color
* Responsive design for mobile and desktop devices

## Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)

## How to Use
1. Open the `Password-generator.html` file in a web browser
2. Click the "Generate Password" button to generate a new password
3. The generated password will be displayed in the text field
4. You can copy the password to your clipboard (if supported by the browser)

## Code Structure
* `Password-generator.html`: The main HTML file that contains the UI elements
* `Password-generator.css`: The CSS file that styles the UI elements
* `Password-generator.js`: The JavaScript file that contains the password generation logic
